A program egy kis családi állatorvosi rendelőnek készült, két orvossal (egy házaspár), valamint egy asszisztenssel. A program célja, hogy le lehessen kérdezni az állatok adatait, előző kórtörténetüket amennyiben van. Az adatok mellett képek mutatják be a különböző fajta állatok növekedési fázisait, valamint a lovak esetében az evolúciós fázisokat. Ez a kiskedvencükkel a rendelőbe látogató gyermekek számára készült elsősorban, hogy kicsit elvonja a figyelmet a beteg állatukról. A program segítségével kiírhatjuk az aktuális napon a betegek és problémáik listáját, orvos avagy faj szerint. Valamint törölni is tudunk, egy adott fajba tartozó minden beteget.
